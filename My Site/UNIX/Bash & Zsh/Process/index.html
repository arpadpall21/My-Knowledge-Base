Command Description
$$ - current shell process id (echo $$ to display)
ps [options] - returns the processes for the current shell (this terminal's pid)
[options]
- (-A|-e) returns all processes
- (aux) returns all processes in an extra detailed list with executable (command) paths
- (-eH) returns all processes as hierarchy tree
- (-e --forest) returns all processes as ASCII hierarchy tree
- (-C processName) returns all processes running under the specified process name
- (--pid|-p pid) returns the specified processes by its pid
- (-U userName) returns all processes belonging to the specified user
- (-G groupName) returns all processes belonging to the specified group
- (-l) detailed list with commands
- (-f) detailed list with executable (command) paths
pgrep [options] processName - returns the pid(s) for the specified process(es) (oldest first)
[options]
- (-v) returns all other processes than the specified one
- (-P ppid) returns all processes belonging to the specified parent process
- (-l) process name included
- (-i) case-insensitive
- (-n) return only the newest process
- (-o) return only the oldest process
- (-c) return a number indicating the number of processes running under the passed process name
|
kill pid ... - kill the specified process(es)
killall [options] processName ... - kill the specified process(es)
[options]
- (-i) case-insensitive
- (-s signal) sends the specified signal instead of SIGTERM
- (-v) report if the signal was successfully sent
- (-l) list avaialble signals
|
top - shows running processes in real time
htop - process manager (advanced version of top)
