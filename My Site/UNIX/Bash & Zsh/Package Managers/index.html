Command Description
APT (Advanced Packaging Tool)
apt install package - installs the specified package
apt remove package - removes (uninstalls) the specified package (does not remove package dependencies and configurations)
apt purge package - removes (uninstalls) the specified package and its configurations (does not remove package dependencies)
apt autoremove - automatically remove all unused dependencies
|
apt update - updates packages on the system (does not upgrades them, only check for available newer versions)
apt upgrade - upgrades dependencies on the system
|
apt list [options] - list all packages on the system (including non installed onces)
[options]
- (--installed) lists installed packages only
- (package) lists the specified package
apt show package - prints informations about the specified package
apt policy package - pinrts the status of the package (installed status, available version, soruce)
|
apt search package - searches for the specified package
|
dpkg
package installer (part of apt)
dpkg -i packagePath - installs the specified pacakge from the local drive
dpkg -r package - removes the specified package (same as apt remove)
dpkg -P package - removes the specified package and its configurations (same as apt purge)
|
dpkg -l package - prints all packages on the system (or the specified one)
|
snap
CLI for Ubuntu App Store (keeps the latest 3 versions)
snap install package - installs the specified snap (package)
snap remove package - removes the specified snap (package) (and its dependencies)
snap revert package - downgrades the specified snap (package) to the previous installed version
snap cleanup - removes all unused packages and dependencies (cache cleanup)
|
snap refresh [options] - upgrades all snaps (packages)
[options]
- (--list) list upgradable packages
|
snap list [options] - lists all enabled packages
[options]
- (--all) lists all packages including disabled onces
- (package) lists the specified snap (package)
|
snap disable package - disables the specified snap (package)
snap enable package - enables the specified snap (package)
|
snap find package - lists some available snaps (packages) or searches for a specified one
|
brew
package manager used on MacOS [link]
brew --version - prints package manager version
brew update - updates homebrew itself to the lastest version
|
brew install package:formulee|cask[@ver] - installs the specified package
brew uninstall | remove | rm package:formulee|cask[@ver] - uninstalls the specified package
brew cleanup - removes all unused packages and dependencies (cache cleanup)
|
brew outdated - lists outdated packages that can be upgraded
brew upgrade - upgrades all installed packages to the latest version
|
brew ls | list - prints installed formulees (packages) and casks (GUI app)
brew info package:formulee|cask[@ver] - prints statistics about the specified package (general info, dependencies, etc..)
|
brew search package:formulee|cask[@ver] - searches the specified package (list several matches including installed onces)
|
