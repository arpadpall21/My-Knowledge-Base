Cloud Concepts

- Cloud Computing -> on-demand delivery of IT resources and applications via the internet with pay-as-you-go pricing

- Elasticity -> Scale up and down as you need

- Agility -> The ability to quickly scale up and down resources

- OpEx - Operational Expenditure (ongoing operational expenses)

- CapEx - Capital Expenditure (upfront capital investment)

- TCO - Total Cost of Ownership

Cloud Computing Models

- IaaS (Infrastructure as a Service) -> service that handles core system parts (hardware, network)

- PaaS (Platform as a Service) -> service that handles software layer parts (operating system, service software, ect...) + Infrastructure layer

- SaaS (Software as a Service) -> service that handles the application (application, data) + Infrastructure layer + Software layer

Deployment Models

- on-premise (private) -> we own our hardware and software

- cloud (public) -> we use public cloud providers like AWS, Azure, etc...

- hybrid -> mix of on-premise and cloud
