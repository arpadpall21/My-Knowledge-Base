String operations [link] [Ordered: Alphabetically]
Command Description
GET keyStr - gets the value of keyStr
SET keyStr val [options] - set val associated with key keyStr
[options]
(EX sec) sets the ttl (expiration value) in seconds
SETEX keyStr sec val - set val associated with key keyStr with sec ttl
SETNX strKey val - set val associated with key keyStr if strKey doesn't exist
- returns 1 on successful operation, otherwise 0
APPEND keyStr val - appends val to the string previously stored with key (creates new if doesn't exist)
MGET keyStr [keyStr ...] - gets multiple strings
MSET keyStr val [keyStr val ...] - sets multiple strings
GETRANGE strKey st(inc):idx end(inc):idx - returns a substring of the range specified by the indexes
SETRANGE strKey offset(inc):idx val - overwrites the strKey value from offset with val
INCR keyStr - increments the value of keyStr by 1 (if the stored value can be interpreted as number)
INCRBY keyStr int - increments the value of keyStr by int (if the stored value can be interpreted as number)
INCRBYFLOAL keyStr float - increments the value of keyStr by float (if the stored value can be interpreted as number)
DECR keyStr - decrements the value of keyStr by 1 (if the stored value can be interpreted as number)
INCRBY keyStr int - decrements the value of keyStr by int (if the stored value can be interpreted as number)
STRLEN strKey - returns the byte length of the stored string
DEL strKey - deletes the specified strKey with its value
- returns 1 on successful deletion, otherwise 0

List operations [link] [Ordered: Alphabetically]
Command Description
LPUSH listKey val ... - pushes elements on the left to the list (creates a new list if not exist)
- returns the number of stored elements
RPUSH listKey val ... - pushes elements on the right to the list (creates a new list if not exist)
- returns the number of stored elements
LINSERT listKey BEFORE|AFTER lsitMember newVal - inserts val in the list BEFORE|AFTER listMember
LPOP listKey - removes and returns the first element in the list
RPOP listKey - removes and returns the last element in the list
LINDEX listKey idx - returns the element at idx
LPOS listKey val - returns the val's index in the list
- returns NIL if no element found
LLEN listKey - returns the list length (if the list doesn't exist returns 0)
LRANGE listKey st(inc):idx end(inc):idx - returns elements in range st(inc) - end(inc) (ex: LRANGE 0 -1 returns all elements)
LREM listKey count val - removes val count times from the list (does not leaves any holes)
- returns the number of elements removed from the list
LTRIM listKey st(inc):idx end(inc):idx - removes all elements that are out of the set range

Set operations [link] [Ordered: Alphabetically]
Command Description
SADD setKey val ... - adds val to the set (creates new if doesn't exist)
SISMEMBER setKey val - returns 1 if val is part of the set, 0 otherwise
SMEMBERS setKey - returns all members of the set
SCARD setKey - returns the number of elements in the set
SMOVE srcSetKeytrgSetKey val - moves val from srcSetKey to trgSetKey
- creates trgSetKey if not exist
- returns 1 on successful operation, otherwise 0
SDIFF setKey setKey ... - returns memebers that are only present in the leftmost set
SINTER setKey setKey ... - returns memebers that are present in all sets
SRANDMEMBER setKey count - returns one (or count) random member(s) from the set
SREM setKey val - removes val from setKey
- returns 1 on successful removal, otherwise 0

SortedSet operations [link] [Ordered: Alphabetically]
Command Description
ZADD sortedSetKey score val score val ... - creates a new sorted set and adds the specified value(s) with its score to it
- returns 1 on successful operation, otherwise 0
ZPOPMAX sortedSetKey [options] - removes and returns the element with the highest score (or count elements)
[options]
- (count) number of elements to pop
ZPOPMIN sortedSetKey [options] - removes and returns the element with the lowest score (or count elements)
[options]
- (count) number of elements to pop
ZSCORE sortedSetKey val - returns the val's score
ZRANGEBYSCORE sortedSetKey stRank(inc) endRank(inc) [options] - returns elements that are scored between the specified range
[options]
- (WITHSCORES) include scores
ZCOUNT sortedSetKey stRank(inc) endRank(inc) - returns the number of elements that are scored between the specified range
ZINCRBY sortedSetKey int|float val - increments the score of val by int|float
- adds val with int|float if member not exist
ZRANK sortedSetKey val - returns the val's index
ZRANGE sortedSetKey st(inc):idx end(inc):idx [options] - returns all elements in the specified index range (ex: ZRANGE 0 -1 returns all elements)
[options]
- (WITHSCORES) include member scores
ZCARD sortedSetKey - returns the number of elements in the sortedSet
ZRANDMEMBER sortedSetKey [options] - returns a random member from the sortedSet
[options]
- (count) number of random elements to return
- (WITHSCORES) include scores
ZREM sortedSetKey val - removes val from the sortedStet
- returns 1 on successful deletion, otherwise 0
ZREMRANGEBYRANK sortedSetKey st(inc):idx end(inc):idx - removes elements between the specified index range
- returns the number of elements removed
ZREMRANGEBYSCORE sortedSetKey st(inc):score end(inc):score - removes elements between the specified score range
- returns the number of elements removed

Hash operations [link] [Ordered: Alphabetically]
Command Description
HGET hashKey key - returns the hash value stored with key
HSET hashKey key val key val ... - sets a new hash with key val pair(s)
HSETNX hashKey key val - sets a new hash with key val pair
- does not override existing key/value pairs
- returns 1 on successful operation, otherwise 0
HMGET hashKey key key ... - returns the hash value stored with key(s)
HMSET hashKey key val key val ... - sets a new hash with key val pair(s)
HGETALL hashKey - returns all key/value pairs stored by hashKey
HKEYS hashKey - returns all the hash keys
HVALS hashKey - returns all the hash values
HEXISTS hashKey key - returns 1 if key exists in hashKey, otherwise 0
HRANDFIELD hashKey [options] - returns a random hash key
[options]
- (count) number of random elements to return
- (WITHVALUES) also return values
HLEN hashKey - returns the number of key/value pairs stored by the hash
HSTRLEN hashKey val - returns the byte length of val
HDEL hashKey key - deletes the hash key
- returns 1 on successful removal, otherwise 0
