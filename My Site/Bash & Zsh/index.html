: gid : groupMember,...)

[options]
- (-n) number output lines
- (-E) highlight output line ends
- (-s) collapses multiple empty lines into a single one
- (>) creates a file (if not exist) and writes the specified content to it (overwrites existing files contents) (cat srcFile > destFile = copy source file content and overwrite target file content)
- (>>) creates a file (if not exist) and appends the specified content to it (appends data to existing files contents) (cat srcFile >> destFile = copy source file content and append it to the target file content)
cd dirPath - navigates to the specified directory
chmod [options] NrNrNr path - changes the permissions of the specified file or direcotry
[options]
- (NrNrNr) (0 = ---) (1 = --x) (2 = -w-) (3 = -wx) (4 = r--) (5 = r-x) (6 = rw-) (7 = rwx)
- (-R) changes the permissions of the targeted directory and all its descendent items
chown [options] ownerName:groupName path - (ownerName) changes the owner of the specified file or directory
[options]
- (:groupName) changes the group ownership of the file (can be written as chown :newGroup file)
- (-R) changes the ownership of the targeted directory and all its descendent items
cp [options] srcPath scrPath ... destPath - copies (and renames) file(s) (cannot copy directories)
[options]
- (-i) prompts (warns) before overwriting files at destination
- (-f) (force) overwrite protected files at destination

- (-s) creates soft-links (symbolic-links) instead of copying files (cannot create directory symbolic-links) (you must be in the directory where you want to create the symbolic-link)

Compression
gzip [-r] [-d] [-c] [-f] [-k] [-(1-9] [-v] [-l] filePath ... - (no options) compresses the specified files by using gzip algorithms, create *.gz files (removes the original file(s) (cannot compress directories)

- (-d) decompresses the specified files
- (-c) the compression result is returned in stdout (printed) (does not modify anyting on the file system)
- (-f) overwrite files at destination
- (-k) keep original file(s) (instead of removing them)
- (-(1-9)) compression level (9 = best compression) (Default: -6)
- (-v) pint compression status at the end of the compression process
- (-l) compression status of the specified compressed file(s)

- the created file should be *.tar

- (-C destDir) target directory where extract files to
tar -cvfz destFile srcPath ... - creates a gzip compressed tarball from the specified items (does not remove original items)
- the created file should be *.tar.gz

- (-C destDir) target directory where extract files to

Wildcards and Globs [Ordered: Alphabetically]
Wildcards Description
Windcards (symbols that stand in for other characters)
* - matches zero or more characters (any character)
? - matches one character (any character)
[char-char] - matches one character (any of the specified range (ex: [a-e] matches a b c d e))
[char...] - matches one character (any of the enclosed character(s) (ex: [aex] matches a e x))
[!char-char] - negative match one character (any other thant the specified range (ex: [!a-e] matches any other character than a b c d e))
[!char...] - negative match one character (any other than the enclosed character(s) (ex: [!aex] matches any other character than a e x))

Globs (pathnames expressed with Wildcards)
/ - segment separator (on all platforms)
* - matches zero or more characters between segment separators
** - matches zero or more characters across segment separators
\\char - escapes the character (on all platforms)
!glob - negative glob, removes one glob from a previous positive glob (must come right under a positive glob)

Remember This :

- wildcard characters (globs) cannot be used in quotes (ex: "ab*") but they can be concatenated (ex: *"ab" | "a"*"b" | "ab"*)

- to change the user and or group id simply modify the user|group id in the /etc/passwd and /etc/group files
