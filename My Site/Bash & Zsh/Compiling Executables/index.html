Command Description
./configure [options] - compiles a software package from source code (responsible for checking the system and environment for the necessary dependencies and tools required to compile the software, it also sets various parameters and options for the build process based on the system configuration, such as specifying the installation directory, enabling or disabling specific features, or selecting the appropriate compiler)
- generates a Makefile based on the system configuration and options specified for the make command
[options]
- (--help) print the help doc (available argumenst)
make [options] - executes the Makefile (in the current directory)
- compiles the software package into an executable or library that then can be installed on the system
[options]
- (--dry-run) prints the result output with actually compiling
|
make install - copies the compiled binaries, libraries, and other files to their appropriate system directories so that they can be accessed by other programs and users
make uninstall - removes binaries, libraries and other files installed with make install (executed from the original install directory)
- this operation works depending on how the original insller was designed
