Environment Variables

Updated ( 2020-04-03 )

MySite> Bash & Zsh> Environment Variables


Command Description
envKey=envVal - sets an environment variable for the current process
export envKey=envVal - sets an environment variable for the current process and its child processes
unset envKey - removes the specified environment varaible
|
$envKey
${envKey} 		- returns the variable's value (ex: echo $envKey | echo ${envKey})
|
${envKey:-defVal} - if envKey is not set returns defVal (default value)
- if envKey is set returns its value
${envKey:+defVal} - if envKey is not set returns an empty string ('')
- if envKey is set returns defVal (default value)
|
env [options] - (no option) returns all environment variables
[options]
- (-0) removes the new line character from the end of each key/value pairs
- (-i cmd) runs the command (program) without environment variables
printenv [options] - reutns all environment variables
[options]
- (-0) removes the new line character from the end of each key/value pairs
