Command Description
gzip [-r] [-d] [-c] [-f] [-k] [-(1-9] [-v] [-l] filePath ... - (no options) compresses the specified files by using gzip algorithms, create *.gz files (removes the original file(s) (cannot compress directories)
- (-r) compress all file(s) in the targeted direcotry (does not compress the directory itself)
- (-d) decompresses the specified files
- (-c) the compression result is returned in stdout (printed) (does not modify anyting on the file system)
- (-f) overwrite files at destination
- (-k) keep original file(s) (instead of removing them)
- (-(1-9)) compression level (9 = best compression) (Default: -6)
- (-v) pint compression status at the end of the compression process
- (-l) compression status of the specified compressed file(s)
|
tar -cvf destFile srcPath ... - creates a uncompressed Linux Archive file (tarball) from the specified items (does not remove original items)
- the created file should be *.tar
tar -rvf tarFile srcPath ... - add items (srcPath) to a *.tar file (tarball) (cannot add items to *.tar.gz and *.tar.tbz files)
tar -xvf tarFile [-C destDir] - extract files from an uncompressed Linux Archive file (tarball) to the curent directory
- (-C destDir) target directory where extract files to
|
tar -cvfz destFile srcPath ... - creates a gzip compressed tarball from the specified items (does not remove original items)
- the created file should be *.tar.gz
tar -xvfz tarGzFile [-C destDir] - uncompresses a gzip compressed tarball to the curent directory
- (-C destDir) target directory where extract files to
|
tar -cvfj destFile srcPath ... - creates a bzip2 compressed tarball from the specified items (does not remove original items)
- the created file should be *.tar.tbz
tar -xvfj tarGzFile [-C destDir] - uncompresses a bzip2 compressed tarball to the curent directory
- (-C destDir) target directory where extract files to
|
tar -tf tarFile - list tarball archived items (works on *.tar | *.tar.gz | *.tar.tbz files)
