commands [Ordered: Alphabetically]
Command Description
nginx [-c filePath] - launches the NGINX daemon
- (-c filePath) launches with the specified config file
nginx -v - prints the version
nginx -t - checks the syntax validity of the default config file and prints its location
nginx -s reload - reloads the configurations without interupting nginx
- the config file with the NGINX was started is reloaded (ex: if started with a custom config, that custom config is reloaded)
- if reloading fails nginx will keep running with the latest config BUT DOES NOT WARN YOU IF THE NEW CONFIGS ARE FAILED TO LOAD!
nginx -s quit - gracefull stop (wait for all worker processes to finish their current request)
nginx -s stop - immediatley kills all nginx processes
